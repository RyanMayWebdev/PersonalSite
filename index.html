Projects

Bubbles

HTML5 | CSS3 | JavaScript | React | Firebase

Bubbles is a project developed to show my understanding of React and firebase and is a realtime chat app. Upon coming to the site the user is presented with a login/sign up screen which uses firebase authentication. Once authenticated, the user is brought to the general chat channel, which will show all messages from the firebase db for general chat. Each message is accompanied by a username and time stamp. The user can also use a dropdown to select other channels. Additionally, I've used the react-input-emoji package to allow users to input emojis into there messages. Messages are also filtered for profanity.

Trivia Blitz

HTML5 | CSS3 | SASS | JavaScript

A paired programming project with Daniel Noordegraaf using the Open Trivia API. The primary focus of this project was to use Javascript and an API to produce a trivia game. The game allows the user to select a category of questions, the difficulty and the amount of questions. This information is used to make the API call, which will return a set of question if they are found, if an error occurs or no results are found the user is notified of this and provided a link to the homepage. Once the API returns data, the game screen appears, presenting the question and 4 answers that are randomized. A timer and score tracker are present, and score is rewarded based on a correct answer and time remaining.

Bar One

HTML5 | CSS3 | SASS

This is a 3 page website that was created based on a provided PSD. Only the desktop view was provided and the mobile view was left for me to decide what would look and function best.

I used only HTML and SASS to develop this website as requested. Each page has it's own partial, combined with a general styling partial and media query partial. This site is also fully responsive and features an animated dropdown mobile menu done through css.

Additionally, accessability was kept in mind the whole way through. The HTML is laid out semantically, all inputs have labels, all images have descriptive alt text and font size was set to 125% as a base or 20px.

Movie Reel

HTML5 | CSS3 | JavaScript | jQuery

A project made to demonstrate working with API's. This site helps users figure out what movie to watch. Enter in a genre and release year or check off new release and then let the code do the work. Alternatively, you can search by full or partial titles. Using jQuery, I get the users input upon submission, make an API call to themoviedatabase and then display the top 20 most popular results to the page. Additionally, each movie will have a description and a movie trailer if one is available, which is also retrieved using the same API to get the movie trailer web address.
